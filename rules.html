Regras do Jogo

  • Objetivo do Jogo: Adivinhar o nome do elemento químico com base no número atômico sorteado de 1 a 50.
  • Início do Jogo: O jogo começa com o computador sorteando um número atômico aleatório entre 1 e 50. O jogador deve tentar adivinhar o nome do elemento correspondente ao número atômico sorteado.
  • Dicas: Se o nome do elemento digitado estiver incorreto, o jogo indicará se o número atômico do elemento sorteado é maior ou menor do que o número atômico do elemento digitado.
  • Tentativas: O jogador tem um total de 6 tentativas para adivinhar o nome do elemento correto. O número de tentativas restantes será exibido após cada tentativa incorreta.
  • Vencedor: O jogador vence se adivinhar o nome do elemento correto dentro das 6 tentativas.
  • Perdedor: O jogador perde se não conseguir adivinhar o nome do elemento correto após 6 tentativas.
